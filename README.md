# InternShield-AI 🛡️

AI Powered Internship Scam Detection System

## Features
✅ Detects fake internship messages  
✅ Scam Risk Score Analysis  
✅ Company Website Check  
✅ Screenshot Upload Support  
✅ AI Advice Generation  
✅ PDF Report Download  

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run the Project

1. Install Python

2. Install required packages:
```bash
pip install flask requests
```

3. Run the project:
```bash
python app.py
```

4. Open browser and visit:
```bash
http://127.0.0.1:5000
```

## Project Purpose
This project helps students identify fake internship scams using AI-based risk analysis and company website checking.
